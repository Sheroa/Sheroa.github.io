跟着慕课网的教程系统学习后的每日小结

第1章 课程介绍

1-1 课程目标和学习内容介绍

什么是前端集成解决方案

  • 解决前端根本问题（通俗说法）
  • 一套包含框架和工具，实现快速构建美丽实用的web应用程序的工作流，同时该工作流必须是稳健强壮的。（学院说法）

    解决哪些前端问题

  • 团队代码风格不统一，如何强制开发规范
  • 前期开发的组件如何使用和维护
  • 如何模块化前端项目
  • 服务器部署前，必须的压缩，检查流程如何简化，流程如何完善

    主流的方式

  • Yeoman bower grunt/gulp
  • Codekit(just for mac)
  • fis(百度) spirit(腾讯)

    grunt的定位

  • build tool

    竞争对手

  • apche ant(曾经大行其道)
  • gulp(新兴)
  • buildy
  • jasy
  • Gmake(jquery原来使用的)

    第2章 准备工作

    2-1 nodejs简介

    nodejs是什么

  • 使得可以用js编写服务器端程序
  • 使得前后端可以使用一种语言全包

    nodejs的安装

  • 官网直接安装
  • 包管理工具安装（HomeBrew for mac）

    命令行的使用

  • iterm for mac
  • npm（nodejs的包管理工具）
  • 几个基本命令

####node项目的基本结构

  • pakage.json
  • node_modules

2-2 yeoman ,bower,grunt简介及安装

安装方法

  • 全是标准npm包。直接npm install -g 包名进行安装

yeoman

  • the web’s scaffolding tool for modern webapps.
  • 用于新建项目的阶段

    bower

  • a package manager for the web
  • 跟踪管理组成web站点的框架、库、公共部分等。

    grunt

  • the javascript task runner
  • build tool
  • 生态太强大，插件多
